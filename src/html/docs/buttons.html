Subtle

Landkit only

Creates a subtle variant of a corresponding contextual button variation. 

<button class="btn btn-primary-subtle">Primary subtle</button>
<button class="btn btn-secondary-subtle">Secondary subtle</button>
<button class="btn btn-success-subtle">Success subtle</button>
<button class="btn btn-danger-subtle">Danger subtle</button>
<button class="btn btn-warning-subtle">Warning subtle</button>
<button class="btn btn-info-subtle">Info subtle</button>

Pills

  Button pill
</button>

Circle

Landkit only

Turn any button into a circle with a .btn-rounded-cirle modifier.
